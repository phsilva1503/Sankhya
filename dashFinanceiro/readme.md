# Projeto Dsh Financeiro

## Escopo do Projeto

Este projeto tem como objetivo o desenvolvimento de um dashboard financeiro para análise e acompanhamento de dados econômicos e financeiros, permitindo uma visualização clara e organizada das informações.

O dashboard irá contemplar indicadores financeiros relevantes, possibilitando o apoio à tomada de decisão, controle de receitas e despesas, análise de desempenho e acompanhamento de resultados ao longo do tempo.
